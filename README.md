# CEP for PR-jsx

A CEP-based Adobe Premiere Pro plugin for quickly loading and running custom JSX scripts.

## Installation and Usage Instructions

1 Before use, run Add Keys.reg first

2 Place your script in the jsx folder. The plugin supports reading scripts from nested folders within the jsx directory, such as ./jsx/cutting and ./jsx/miscellaneous

3 Click the button to run the script
